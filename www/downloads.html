Ubuntu

RetroShare is currently available on Ubuntu Jaunty, Karmic, Lucid, Maverick, Natty, Oneiric and Precise.

To install RetroShare on ubuntu, please use one of the following two PPA repositories:
Link: retroshare
Link: retroshare-snapshots

# for RetroShare releases only
   sudo add-apt-repository ppa:csoler-users/retroshare
# for RetroShare development snapshots
   sudo add-apt-repository ppa:csoler-users/retroshare-snapshots
# then
   sudo apt-get update
   sudo apt-get install retroshare
RetroShare will be available in Applications->Internet

Important note for users of Ubuntu Natty+Unity:

Unity will hide RetroShare's systray icon by default, which makes it impossible to bring back once you have closed its main window. To prevent this behavior, you need to:

  • Install the dconf-tools package
  • Launch dconf-editor, go to Desktop -> Unity -> Panel, and add 'RetroShare' to the systray-whitelist variable
