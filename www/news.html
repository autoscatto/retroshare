November 14th 2007 - RetroShare 0.3.52a released

Improvements:.

changelog:

  • New improved search system
  • New Translations
  • Can now remove Shared Directories.
  • Attached the 'Recommend to' Context Menus
  • New Timestamps in Chats / Messages
  • Notification of Offline Peer in Chats
  • Example Games (not networked yet)
  • Updated About Text.

October 02th 2007 - RetroShare 0.3.50a released

We are pleased to announce the latest and greatest Retroshare - V0.3.50a. This release brings significant improvements in terms of file sharing.

changelog:

  • File Cache - your peers directories are viewable even when they are offline.
  • File Hashing - downloads are now sourced from any online peer which has the file.
  • File Search - the new file cache enables instant keyword searching through all available files
  • Improved stability

    • This release is only partially compatible with the previous version (v0.3.12b), so we recommend that everyone upgrades to the new version.
