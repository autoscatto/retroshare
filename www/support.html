User support - Help

User support is provided by the RetroShare users and developers community .

You can find resources and informations on how to get support and help for RetroShare software.
We will do our best.

Gettings Help

FAQ

If you have any questions about RetroShare, you should always start by reading the Official FAQ.

HOW TO

If you have a question about how to do something with RetroShare, you should:

  • read the Documentation,
  • browse our wiki.

Problems

If you are experimenting problems using RetroShare, you should:

  • read the Common problems page in our wiki.
  • search the official forum for an answer.

Ask a question

If this is not enough, which can happen, ask a question. For that, you need to get information and prepare a correct report for people to help you.
Our Wiki will help you to do it.

Then ask your question, on:

  • the Forum, in the correct section;
  • the users mailing-lists,
  • the IRC channel on the freenode Network (irc://irc.freenode.net, #retroshare), for live answers. Use the web-based IRCWeb interface, if you need it.

Developers and Reporting bugs

Developers

You are a developer and you want to help the project or build a project around our software?

Read the developers section of this website or go and finds info on the Developer's corner. You should also use the developers mailing-lists and IRC.

Report a bug

Read and conform yourself to our Reporting Bugs policy and look on our Bug tracker.

Contacting the team

If your matter is not answered yet, and you need to contact the team, do it by mail, But please prefer the users mailing-list first for support .

We have a lot of spam and lots of e-mail, be sure to state in the subject that you want to speak about RetroShare. Try to not write HTML mails, thanks.
